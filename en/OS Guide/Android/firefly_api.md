# Overview 
FireflyApi provides part of the system interface and encapsulates the functional interfaces required by some users, mainly to allow users to easily and simply use the common interfaces of the system.

For details, please refer to [WiKi Tutorial](https://wiki.t-firefly.com/en/FireflyApi/FireflyApi.html)。
