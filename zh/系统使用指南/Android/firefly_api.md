# 概述
FireflyApi提供了部分系统接口以及封装了部分用户需要的功能接口，主要是为了让用户容易和简单的使用系统常用接口。

详细参考[WiKi教程](https://wiki.t-firefly.com/zh_CN/FireflyApi/FireflyApi.html)。
